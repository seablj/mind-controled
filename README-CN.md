## 简述

**如果这个库对你有帮助，请给作者一颗:star:**<br>

目前它只是一个物联网通信的工具，包含

- 西门子S7通信协议
- ModbusTCP通信协议
- RTSP、RTCP、RTP、H264协议
- MP4 (FMP4) 协议
- 基础字节数组解析转换工具

## 使用指南

在JAVA工程的pom.xml文件中添加该依赖即可

```xml

<dependency>
    <groupId>com.github.xingshuangs</groupId>
    <artifactId>iot-communication</artifactId>
    <version>1.4.0</version>
</dependency>
```
